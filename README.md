# Proyek-Akhir-Basis-Data---Klik-dan-Tonton---Sistem-Manajemen-Penjadwalan-TV
Klik dan Tonton adalah aplikasi berbasis web yang bertujuan untuk mempermudah pengguna dalam mengakses jadwal siaran TV digital dan manajemen penjadwalan acara. Aplikasi ini mencakup pengelolaan data pengguna, saluran TV, serta transaksi penempatan iklan, dengan sistem database yang terstruktur dan efisien.
